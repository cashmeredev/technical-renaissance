#+title: Technical Renaissance Webring

* Introduction

This is the official webring for the =#technicalrenaissance= IRC channel, created by [[https://joshblais.com][Joshua Blais]].

* How to join

Add the following HTML code somewhere at your website:

#+begin_src html
<a href="https://your-ring-host/prev?from=yourname">← previous</a>
<a href="https://your-ring-host/random">random</a>
<a href="https://your-ring-host/next?from=yourname">next →</a>
#+end_src

#+begin_note
It's usually placed at the footer
#+end_note

Fork this repository, add yourself to the [[file:config.toml][config.toml]] and create a new PR.

Feel free to subscribe to our combined. All you have to do is add the link to your favorite RSS reader.
